#  JavaScript Calculator

A simple and responsive calculator built using HTML, CSS, and JavaScript.

##  Features
- Basic arithmetic operations (+, -, ×, ÷ , **, %)
- Clear all (C) and delete last digit (⌫)
- Responsive UI
- Clean and modern design

##  Tech Stack
- HTML  
- CSS  
- JavaScript
